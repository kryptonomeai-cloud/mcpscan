## New Releases

### Holotron-12B (H Company × NVIDIA) — Mar 17
- **Type:** Open-weight multimodal computer-use agent
- **Base:** Post-trained from NVIDIA Nemotron-Nano-2 VL
- **Architecture:** Hybrid SSM + attention (reduced KV cache, constant state per layer)
- **Key capability:** Desktop/web navigation with 2× throughput over Holo2-8B on single H100
- **Benchmark:** Strong on WebVoyager; scales efficiently to 8.9k tokens/s at concurrency 100
- **Relevance:** 12B model could potentially run on our 3090 setup for local computer-use agents
- **Link:** https://huggingface.co/blog/Hcompany/holotron-12b

### NVIDIA SPEED-Bench — Mar 19
- New unified benchmark for evaluating speculative decoding across diverse semantic domains
- Two splits: Qualitative (draft accuracy across domains) + Throughput (system-level speedups at various ISL/concurrency)
- Relevant for anyone optimizing local inference with draft models
- **Link:** https://huggingface.co/blog/nvidia/speed-bench

## Updates to Existing Models

### OpenAI GPT-5.3 Instant (rolling out now)
- New default ChatGPT model replacing GPT-5.2 Instant
- Hallucination improvements: ↓26.8% for web search, ↓19.7% for internal knowledge
- Less robotic/preachy writing style
- Trade-off: Slightly worse safety scores vs GPT-5.2 Instant, small HealthBench regression
- API: `gpt-5.3-chat-latest`
- GPT-5.2 Instant remains for paid users until June 3, 2026
- **Source:** https://the-decoder.com/openai-releases-gpt-5-3-instant-for-smoother-everyday-conversations-and-better-search/

### OpenAI ChatGPT Model Picker Redesign
- Three tiers: **Instant** (quick), **Thinking** (complex), **Pro** (most powerful)
- Dropdown for specific versions: Latest (5.4), 5.2, 5.0, o3
- "Configure" menu adds old Auto mode (routes between Instant/Thinking)
- New "Nerdy" personality style added
- **Source:** https://the-decoder.com/
