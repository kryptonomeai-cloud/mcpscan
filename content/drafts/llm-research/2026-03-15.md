## New Releases (past ~1-2 weeks)

### NVIDIA Nemotron 3 Super 120B-A12B
- **Released:** March 11, 2026
- **Architecture:** LatentMoE — Mamba-2 + MoE + Attention hybrid with Multi-Token Prediction (MTP)
- **Params:** 120B total, 12B active
- **Context:** Up to 1M tokens
- **Languages:** EN, FR, DE, IT, JA, ES, ZH
- **Key benchmarks:** AIME25 90.2%, RULER@1M 91.75%, SWE-Bench (OpenHands) 60.5%, MMLU-Pro 83.7%
- **Min hardware:** 8× H100-80GB (FP4 version runs on single B200 or DGX Spark)
- **License:** NVIDIA Nemotron Open Model License (commercial OK)
- **Optimized for:** Agentic workflows, long-context reasoning, RAG, tool use

### MiniMax M2.5
- **Released:** ~March 10, 2026
- **Params:** 229B (MoE)
- **Key benchmarks:** SWE-Bench Verified 80.2%, Multi-SWE-Bench 51.3%, BrowseComp 76.3%
- **Speed:** 100 TPS (M2.5-Lightning) or 50 TPS (M2.5 standard)
- **Pricing:** $0.15-0.30/M input, $1.20-2.40/M output — claims 10-20× cheaper than Opus/GPT-5/Gemini 3 Pro
- **Notable:** Trained with RL in 200K+ real-world environments. Spec-writing behavior emerged (plans before coding). SWE-Bench completion 37% faster than M2.1, matching Claude Opus 4.6 speed at 10% cost.

### Sarvam-105B
- **Released:** ~March 10, 2026
- **Architecture:** MoE, 10.3B active params, MLA attention
- **Context:** 128K (YaRN scaling)
- **Strengths:** Agentic tasks (τ² Bench avg 68.3%), BrowseComp 49.5%, AIME25 88.3% (96.7% with tools)
- **Focus:** Indian languages (22 languages), Apache 2.0 license
- **Also released:** Sarvam-30B (smaller variant)

### Reka Edge (reka-edge-2603)
- **Released:** ~March 13, 2026
- **Size:** 7B multimodal VLM
- **Focus:** Vision tasks, edge deployment (runs on Jetson, phones, Apple Silicon)
- **Key:** Uses only 331 tokens for 1024×1024 image (vs 1000+ for competitors), 2-3× faster latency
- **Benchmarks:** VQA-v2 88.4%, MLVU 74.3%, Mobile Actions 88.4%
- **License:** Commercial if <$1M revenue

### Tesslate OmniCoder-9B
- **Released:** ~March 13, 2026
- **Base:** Qwen3.5-9B (hybrid Gated Delta Networks + attention)
- **Training:** LoRA SFT on 425K agentic coding trajectories from Claude Opus 4.6, GPT-5.4, GPT-5.3-Codex, Gemini 3.1 Pro
- **Benchmarks:** GPQA Diamond 83.8% (+2.1 over base), Terminal-Bench 2.0 23.6% (+61% over base)
- **License:** Apache 2.0

### HumeAI TADA (TTS)
- **Released:** ~March 14, 2026
- **Models:** tada-1b (Llama 3.2 1B base), tada-3b-ml (Llama 3.2 3B base)
- **Innovation:** 1:1 text-acoustic alignment — each text token maps to exactly one speech vector. Dynamic duration synthesis eliminates fixed frame rates. Eliminates transcript hallucination.

### Fish Audio S2 Pro (TTS)
- **Released:** ~March 12, 2026
- **Size:** ~5B (4B slow AR + 400M fast AR)
- **Training:** 10M+ hours audio, 80+ languages
- **Innovation:** Fine-grained inline prosody/emotion control via [tag] syntax (15K+ unique tags). RL-aligned. SGLang-based streaming inference.

### IBM Granite 4.0-1b-speech
- **Released:** March 6, 2026
- **Size:** 1B (compact ASR/AST)
- **Languages:** EN, FR, DE, ES, PT, JA
- **Notable:** Half the params of granite-speech-3.3-2b, keyword biasing for name/acronym recognition
- **License:** Apache 2.0

## Relevance to Us (OpenClaw on Mac mini + 5× RTX 3090 GPU server)

### Worth Testing
1. **OmniCoder-9B** — 9B model, easily runs on a single 3090. Apache 2.0. Distilled from frontier coding traces. Could be excellent for local coding assistance. GGUF available.
2. **Reka Edge 7B** — Runs on Apple Silicon (24GB+). Excellent vision model for edge. Could run on Mac mini for local image/video understanding.
3. **Qwen3.5-9B / 35B-A3B** — The 9B fits one 3090 easily; 35B-A3B (MoE, ~3B active) is incredibly efficient. Both trending heavily. GGUF available via unsloth.
4. **TADA TTS (1B)** — Tiny, could run locally for speech synthesis with novel alignment approach.

### Worth Watching
5. **MiniMax M2.5 via API** — If pricing holds, this could be the most cost-effective frontier model for agentic OpenClaw tasks. Worth benchmarking against Claude for our use cases.
6. **Nemotron 3 Super** — Too large for our hardware (needs 8×H100 at BF16), but FP4 version could potentially run on 5×3090 with some effort. 1M context is compelling.
7. **Sarvam-105B** — MoE with 10.3B active, could theoretically fit on our 3090 cluster with offloading. Apache 2.0.

### Skip for Now
- Fish S2 Pro (5B TTS) — Interesting but we have ElevenLabs via sag
- Granite 4.0-1b-speech — Niche ASR, we have better options
- LocoTrainer-4B — Too specialized (MS-SWIFT framework only)
