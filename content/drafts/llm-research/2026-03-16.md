## New Releases

### Hume AI TADA (open-sourced ~March 10–13)
- **What**: Speech language model with 1:1 text-to-audio token alignment
- **Sizes**: 1B (English) and 3B (8 languages) — both based on Llama 3.2
- **Key innovation**: Each autoregressive step covers one text token, dynamically determining duration and prosody. Zero transcript hallucinations across 1,000+ test samples.
- **Speed**: 5× faster than comparable systems, compact enough for smartphones
- **License**: MIT
- **Links**: [GitHub](https://github.com/HumeAI/tada) | [HF](https://huggingface.co/collections/HumeAI/tada)
- **Relevance**: Could run the 1B model on a single 3090 easily. Good for local TTS.

### IBM Granite 4.0 1B Speech (March 9)
- **What**: Compact multilingual ASR/AST model, half the params of predecessor (granite-speech-3.3-2b)
- **Languages**: English, French, German, Spanish, Portuguese, Japanese (new)
- **Features**: Keyword list biasing for names/acronyms, #1 on OpenASR leaderboard
- **License**: Apache 2.0
- **Relevance**: Tiny enough to run anywhere. Good for edge speech-to-text.

## Updates to Existing Models

### Anthropic: 1M Context Window GA (March 13)
This is the biggest news of the past few days:
- **Models**: Claude Opus 4.6 and Claude Sonnet 4.6
- **Pricing**: Standard rates across the FULL 1M window — no long-context premium
  - Opus 4.6: $5 in / $25 out per million tokens (same at 9K or 900K)
  - Sonnet 4.6: $3 in / $15 out per million tokens
- **Media limits**: 600 images or PDF pages per request (up from 100)
- **No beta header required** — requests over 200K tokens work automatically
- **Claude Code**: Max, Team, Enterprise users on Opus 4.6 get 1M context by default — fewer compactions
- **Benchmark**: Opus 4.6 scores 78.3% on MRCR v2 — highest among frontier models at 1M context
- **Available on**: Claude Platform, Azure Foundry, Vertex AI, Amazon Bedrock

**Key pricing insight** (via Simon Willison): OpenAI charges more for GPT-5.4 above 272K tokens, Google charges more for Gemini 3.1 Pro above 200K tokens. Anthropic now charges the same rate across the entire window. This makes Anthropic significantly cheaper for long-context workloads.

## Open Source

### NVIDIA NeMo Retriever Agentic Pipeline (March 13)
- **#1 on ViDoRe v3 pipeline leaderboard**, #2 on BRIGHT (reasoning-intensive retrieval)
- Uses ReACT architecture with iterative search-evaluate-refine loop
- Agent uses `think`, `retrieve(query, top_k)`, and `final_results` tools
- Key insight: generalizable design beats dataset-specific heuristics
- Falls back to Reciprocal Rank Fusion when hitting step/context limits

### Ai2 MolmoBot & MolmoSpaces (March ~11)
- Robotics models trained entirely in simulation — zero-shot sim-to-real transfer
- MolmoSpaces: 230K+ indoor scenes, 130K+ objects, 42M+ grasping annotations
- MolmoBot: picks up objects, opens drawers, operates doors — no real training data
- All models and tools openly available

### HuggingFace Storage Buckets (March 10)
- S3-like mutable object storage on the Hub (non-versioned)
- Built on Xet (chunk-based deduplication) — efficient for ML checkpoints
- Useful for training runs, data pipelines, agent traces

### Autoresearch Pattern (via Shopify/Tobi Lütke, March 13)
- Shopify CEO used Pi agent + autoresearch pattern to optimize Liquid template engine
- 93 commits from ~120 automated experiments → 53% faster parse+render, 61% fewer allocations
- Pattern: agent brainstorms improvements, experiments one at a time against benchmarks
- Key takeaway: robust test suites are massive unlocks for coding agents

## Benchmarks & Comparisons

- **MRCR v2 (long-context retrieval)**: Opus 4.6 at 78.3% — highest among frontier models at 1M tokens
- **ViDoRe v3 pipeline**: NVIDIA NeMo Retriever #1
- **BRIGHT (reasoning retrieval)**: NVIDIA NeMo Retriever #2
- **OpenASR**: IBM Granite 4.0 1B Speech #1 among open speech models
- **LLM Neuroanatomy** (trending on r/LocalLLaMA): Researcher discovered you can improve model performance by duplicating middle layers (the "reasoning" layers) without changing any weights. Topped HuggingFace Open LLM Leaderboard with RYS-XLarge (72B + duplicated layers). Fascinating research on transformer internal structure.

## Pricing Changes

### Anthropic 1M Context — Flat Rate (March 13)
| Model | Input (per M) | Output (per M) | Long-Context Premium |
|-------|--------------|----------------|---------------------|
| Claude Opus 4.6 | $5 | $25 | **None** (same rate full window) |
| Claude Sonnet 4.6 | $3 | $15 | **None** |
| GPT-5.4 (OpenAI) | Varies | Varies | Higher above 272K tokens |
| Gemini 3.1 Pro (Google) | Varies | Varies | Higher above 200K tokens |

Anthropic's move eliminates the "long-context tax" entirely. For workloads regularly hitting 200K+ tokens (which is common in agentic/coding scenarios), this could meaningfully reduce costs.
