## New Releases

### OpenAI GPT-5.4 mini and GPT-5.4 nano (March 17)
Two weeks after GPT-5.4 landed (March 5), OpenAI fills out the family:

**GPT-5.4 mini:**
- 400K context window
- $0.75 / $4.50 per M tokens (input/output), $0.075 cached input
- 2× faster than GPT-5 mini
- SWE-Bench Pro: 54.4% (vs 45.7% for GPT-5 mini, 57.7% for full GPT-5.4)
- OSWorld-Verified: 72.1% (approaches GPT-5.4's 75.0%)
- GPQA Diamond: 88.0%
- Available in API, Codex (30% of GPT-5.4 quota), ChatGPT (Free/Go via Thinking menu)
- Supports text+image inputs, tool use, function calling, web search, computer use

**GPT-5.4 nano:**
- API only, $0.20 / $1.25 per M tokens
- Cheaper than Gemini 3.1 Flash-Lite ($0.25/$1.50)
- SWE-Bench Pro: 52.4% — still outperforms GPT-5 mini
- Best for: classification, data extraction, ranking, coding subagents
- Simon Willison tested: can describe his entire 76K photo collection for ~$52

**Subagent pattern:** OpenAI is pushing a hub-spoke model where GPT-5.4 plans/coordinates while mini/nano handle subtasks in parallel. Codex natively supports this.

### NVIDIA GTC 2026 Announcements (March 16-19, ongoing)

**Hardware:**
- **Vera Rubin Platform**: Full-stack computing platform — 7 chips, 5 rack-scale systems, 1 supercomputer for agentic AI. Ships later 2026. 10× performance/watt vs Grace Blackwell. 1.3M components per system.
- **Groq 3 LPU**: NVIDIA's first chip from acquired Groq ($20B deal, Dec 2025). Groq 3 LPX rack holds 256 LPUs alongside Vera Rubin. Claims 35× tokens/watt improvement.

- **DLSS 5**: AI-powered 3D-guided neural rendering for real-time photoreal 4K on local hardware.

**AI/Software:**
- **NemoClaw**: Security/privacy wrapper for OpenClaw (teased March 17)
- Jensen projects $1 trillion in Blackwell + Vera Rubin revenue through 2027
- NVIDIA shares ~$4.5T market cap, 11 straight quarters of 55%+ revenue growth

### Hunter Alpha — Mystery 1T-Parameter Model (March 11, ongoing)
- Appeared anonymously on OpenRouter March 11 with no attribution
- 1 trillion parameters, 1 million token context window
- Describes itself as Chinese AI model with May 2025 training cutoff (matches DeepSeek)
- 160+ billion tokens processed in one week — massive adoption despite anonymity
- Reasoning style analysis points toward DeepSeek training methods
- Skeptics cite architectural differences from known DeepSeek systems

- **If confirmed as DeepSeek V4**: Would be their largest model yet, free to use, potentially game-changing

## Relevance to Us (Mac mini + 5× RTX 3090)

### Immediate Actions
1. **GPT-5.4 nano for OpenClaw subagent tasks** — At $0.20/$1.25, this is excellent for classification, data extraction, and lightweight coding subagents. Consider as default for simple agentic subtasks.
2. **GPT-5.4 mini as mid-tier option** — $0.75/$4.50 with 400K context is strong value. Good fallback when Opus/Sonnet is overkill but nano isn't enough.
3. **Test Gemini Built-in Tools + Function Calling combo** — If we use Gemini for search-grounded tasks, the new combined feature could reduce round trips.

### Watch This Week
- **Hunter Alpha / DeepSeek V4** — If confirmed and open-weighted, a 1T model with 1M context could be transformative. Watch for April announcement.
- **NVIDIA NemoClaw** — Security wrapper for OpenClaw. Directly relevant to our setup. Details expected from remaining GTC sessions.
- **Gemini 3.1 cost controls** — Monthly spending caps in AI Studio could help budget management.

### No Changes Needed
- Our local model stack (Qwen3.5, Mistral Small 4 NVFP4) remains solid. No new open-source releases this cycle that change the local picture.
- Claude Opus 4.6 / Sonnet 4.6 still our primary API models — no competitor has clearly surpassed them for complex reasoning.
