## TL;DR

- **Xiaomi MiMo-V2-Pro** launched — 1T-param sparse model (42B active), #10 on Artificial Analysis Intelligence Index, approaching GPT-5.2/Opus 4.6 at ~1/6th the cost. Open-source variant planned.
- **MiniMax M2.7** released — proprietary reasoning model that performed 30-50% of its own RL training workflow. 66.6% medal rate on MLE Bench Lite (ties Gemini 3.1). Chinese labs pivoting to proprietary models.
- **Cursor Composer 2** launched — fine-tuned Kimi K2.5 for long-horizon agentic coding. Beats Opus 4.6 on Terminal-Bench 2.0 (61.7 vs 58.0). 86% cheaper than Composer 1.5. GPT-5.4 still leads at 75.1.
- **Mamba 3** released under Apache 2.0 — inference-first SSM architecture, ~4% improvement over transformers in language modeling, half the state size of Mamba-2. Could reshape efficient inference.
- **Nvidia KVTC** — new technique shrinks KV cache memory 20× without changing model weights, 8× latency reduction. Huge for long-context serving.

## New Releases

### Xiaomi MiMo-V2-Pro
- **Provider:** Xiaomi (led by ex-DeepSeek researcher Fuli Luo)
- **Parameters:** 1T total, 42B active (sparse MoE)
- **Context:** 1M tokens
- **Architecture:** Hybrid Attention (7:1 ratio), Multi-Token Prediction
- **Benchmarks:** #10 Artificial Analysis Intelligence Index (score 49), GDPval-AA Elo 1426 (ahead of GLM-5's 1406), same tier as GPT-5.2 Codex
- **Hallucination rate:** 30% (vs MiMo-V2-Flash's 48%)
- **Pricing:** API available, ~1/6th to 1/7th the cost of GPT-5.2/Opus 4.6 for <256K token requests
- **Open source:** Variant planned "when stable enough"
- **Relevance:** Significant Chinese frontier model. Not yet open-weight so API-only for now. Watch for the open release.

### MiniMax M2.7
- **Provider:** MiniMax (Chinese startup)
- **Type:** Proprietary reasoning model
- **Key innovation:** Self-evolving — handled 30-50% of its own RL development workflow
- **Benchmarks:** SWE-Pro 56.22% (matching GPT-5.3-Codex), GDPval-AA Elo 1495, MLE Bench Lite 66.6% medal rate
- **Hallucination rate:** 34% (lower than Sonnet 4.6's 46% and Gemini 3.1 Pro's 50%)
- **Significance:** Second Chinese lab going proprietary (after z.ai/GLM-5 Turbo). Self-improvement loop is notable.

### Cursor Composer 2
- **Provider:** Anysphere/Cursor
- **Base:** Fine-tuned Kimi K2.5
- **Context:** 200K tokens
- **Pricing:** Standard $0.50/$2.50 per M tokens; Fast $1.50/$7.50 (86% cheaper than Composer 1.5)
- **Benchmarks:** CursorBench 61.3 (vs 44.2 for 1.5), Terminal-Bench 2.0 61.7 (beats Opus 4.6's 58.0, trails GPT-5.4's 75.1), SWE-bench Multilingual 73.7
- **Availability:** Cursor-only, not a standalone API
- **Relevance:** Shows Kimi K2.5 as a strong fine-tuning base. Cursor now less dependent on external models.

### Mamba 3
- **Researchers:** Albert Gu (CMU), Tri Dao (Princeton)
- **License:** Apache 2.0
- **Scale:** 1.5B parameter research release
- **Key claim:** ~4% relative improvement over transformers in language modeling at same scale, half the state size of Mamba-2
- **Design philosophy:** "Inference-first" — optimized for GPU utilization during decoding, not just training throughput
- **MIMO variant:** 57.6% average accuracy across benchmarks (+2.2 points over standard transformer)
- **Relevance:** Architecture research, not a production model yet. But as hybrid Mamba models proliferate (Nemotron 3 Super, etc.), this is the foundation.
