## New Releases

### Mistral Small 4 (119B-2603) — Released ~March 24
- **Architecture:** MoE, 128 experts, 4 active per token = **6.5B active params** out of 119B total
- **Context:** 256k tokens
- **Modality:** Text + Image input, text output
- **Capabilities:** Unified instruct + reasoning model (reasoning_effort toggle). Native function calling, JSON output, agentic. Multilingual (12+ languages)
- **License:** Apache 2.0
- **Performance:** Competitive with GPT-OSS 120B on AIME, LiveCodeBench, AA-LCR benchmarks while producing significantly shorter/more efficient outputs (1.6K chars vs 5.8-6.1K for Qwen models on AA-LCR)
- **Deployment:** vLLM recommended; also llama.cpp (GGUFs from Unsloth), LM Studio, SGLang, transformers
- **Speculative decoding:** Trained eagle head available (`Mistral-Small-4-119B-2603-eagle`)
- **Quantised:** NVFP4 checkpoint available
- **Source:** https://huggingface.co/mistralai/Mistral-Small-4-119B-2603

### NVIDIA Nemotron-Cascade-2-30B-A3B — Updated March 24-25
- **Architecture:** MoE, 30B total, 3B active params
- **Post-trained from:** Nemotron-3-Nano-30B-A3B-Base
- **Context:** 262k tokens
- **Highlights:**
  - 🥇 IMO 2025 Gold Medal (35 pts)
  - 🥇 IOI 2025 Gold Medal (439.3 pts)
  - 🥇 ICPC World Finals 2025 (10/12)
  - AIME 2025: 92.4 (98.6 with tools)
  - LiveCodeBench v6: 87.2
  - ArenaHard v2: 83.5 avg
- **License:** NVIDIA Nemotron Open Model License (commercial use OK)
- **Deployment:** vLLM ≥ 0.17.1, single GPU possible
- **Source:** https://huggingface.co/nvidia/Nemotron-Cascade-2-30B-A3B

### NVIDIA Nemotron-3-Super-120B-A12B — Released March 11 (still trending)
- **Architecture:** LatentMoE — Mamba-2 + MoE + Attention hybrid with Multi-Token Prediction
- **Params:** 120B total, 12B active
- **Context:** Up to 1M tokens
- **Best for:** Agentic workflows, long-context reasoning, high-volume workloads
- **Reasoning mode:** Configurable on/off
- **Minimum HW:** 8× H100-80GB (NVFP4 for single B200/DGX Spark)
- **Notable benchmarks:** SWE-Bench Verified (OpenHands): 60.47, AIME 2025: 90.21, RULER@1M: 91.75
- **Source:** https://huggingface.co/nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-BF16

## Relevance to Us (Mac Mini + 5× RTX 3090 GPU Server)

### 🔥 High Priority — Try These
1. **Nemotron-Cascade-2-30B-A3B** — Only 3B active params, runs on **single GPU**. IMO gold-medal reasoning. Should run great on one RTX 3090 with GGUF quants. Strong coding + math.

2. **Qwen3.5-27B-Claude-4.6-Opus-Reasoning-Distilled** — Community favourite, tested specifically on RTX 3090. Claude-quality reasoning in a local 27B package. GGUFs available. Excellent for coding agents.

3. **Qwen3.5-9B-Claude-4.6-Opus-Reasoning-Distilled-v2** — Even smaller, runs easily on single 3090. Good coding-agent performance with developer role support.

### 📋 Worth Watching
4. **Mistral Small 4 (119B)** — Apache 2.0 but needs more VRAM than we have for full precision. With 6.5B active params though, GGUF quants might work across multiple 3090s. The NVFP4 checkpoint is promising but designed for NVIDIA H100/B200.

5. **Nemotron-3-Super-120B-A12B** — Minimum 8×H100-80GB so not runnable on our setup. But the NVFP4 version might be interesting if llama.cpp/exllamav2 quantizations appear.
