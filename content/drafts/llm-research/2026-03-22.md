## New Releases

### Cursor Composer 2 (Mar 19)
- **Provider:** Cursor (fine-tuned Kimi K2.5 via Fireworks AI commercial partnership)
- **Pricing:** $0.50/M input, $2.50/M output (standard); $1.50/$7.50 (fast variant, now default)
- **Benchmarks:**
  - CursorBench: 61.3 (vs 44.2 Composer 1.5, 38.0 Composer 1)
  - Terminal-Bench 2.0: 61.7 (vs 47.9 / 40.0)
  - SWE-bench Multilingual: 73.7 (vs 65.9 / 56.9)
- **Key:** First continued pretraining run + RL on long-horizon coding. Self-summarization enables training on trajectories longer than context window.
- **Source:** https://cursor.com/blog/composer-2

### Flash-MoE: Running Qwen 397B on 48GB MacBook
- Dan Woods got Qwen3.5-397B-A17B running at 5.5+ t/s on M3 Max 48GB
- Used Apple's "LLM in a Flash" paper techniques via Claude Code autoresearch
- 2-bit expert quantization (upgraded to 4-bit for tool calling reliability)
- Non-expert weights (embedding table, routing matrices) kept at full precision: 5.5GB resident
- Code: https://github.com/danveloper/flash-moe
- **Relevance:** Technique potentially applicable to our 5× RTX 3090 setup for giant MoE models
