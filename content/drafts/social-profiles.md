### First Post (Launch Announcement)
🛡️ Excited to announce the launch of MCPScan — the first open-source security scanner for MCP (Model Context Protocol) servers.

As AI agents become critical infrastructure, the tools they connect to need proper security review. MCPScan analyses your MCP server configurations and tool definitions to find SSRF vulnerabilities, injection vectors, authentication gaps, and excessive permissions — all locally, with zero cloud dependencies.

Why this matters: Microsoft recently disclosed a critical SSRF vulnerability in their own Azure MCP server. If one of the world's largest cloud providers can ship insecure MCP implementations, it's clear the ecosystem needs better tooling.

MCPScan is open source, MIT licensed, and takes seconds to run:

pip install mcpscan
mcpscan scan --config your-config.json

We'd love your feedback. What MCP servers is your team running? What security concerns keep you up at night?

GitHub: github.com/kryptonomeai-cloud/mcpscan
