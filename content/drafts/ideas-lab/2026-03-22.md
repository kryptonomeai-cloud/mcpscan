## 1. Infrastructure & Automation

### Top Idea: Deploy Dockhand as Unified Docker Management Layer

**Dockhand** has exploded in the self-hosted community, effectively replacing 7 separate Docker tools (Portainer, Lazydocker, ctop, etc.) in a single, blazing-fast interface. It's self-hosted only (no SaaS), includes vulnerability scanning, and is free for personal/internal business use.

**Why this matters now:** You're running Docker across Mac mini, GPU server, and NAS. Managing containers across three hosts with separate tools is friction. Dockhand gives you one dashboard with container monitoring, log tailing, shell access, resource usage, and vulnerability scanning — across all hosts.

**Revenue Potential:** Not direct revenue, but operational efficiency. Fewer tools = fewer things to maintain, less context-switching, faster incident response. Time saved → more time on revenue-generating work.

### Quick Win: Deploy Floci as LocalStack Replacement

**Floci** (70 pts on HN today) is a free, open-source local AWS emulator written in Java/GraalVM native binary. LocalStack's community edition just sunset in March 2026 — requiring auth tokens, dropping CI support, and freezing security updates. If you're doing any AWS development or testing, Floci is the new go-to. It's a single binary, no Docker required, and covers S3, SQS, SNS, DynamoDB, Secrets Manager, and more. `docker run -p 4566:4566 hectorvent/floci:latest` — 5 minutes.

### Notable Finds

1. **Grafeo — Embeddable Graph Database in Rust** (191 pts HN) — Pure Rust, no C dependencies, bindings for Python/Node.js/Go/WASM. Can be embedded as a library OR run standalone. Relevant for knowledge graphs, RAG applications, or relationship mapping in your security consulting work. The WASM bindings mean you could run it client-side in a browser.

2. **Tooscut — Browser Video Editing via WebGPU + WASM** (145 pts HN) — Professional video editing entirely in the browser using Rust compiled to WASM with WebGPU acceleration. Open source. If you're producing cybersecurity content, this eliminates the need for DaVinci Resolve or Premiere for quick edits. Zero install.

3. **Self-Host Weekly (20 March 2026) highlights:** Unraid v7.3.0 beta (install OS on non-flash drives finally); Google walked back Android sideloading restrictions; NanoClaw's creator scored a deal with Docker; AI-generated PRs are overwhelming open source maintainers (Node.js had to publish AI contribution guidelines after a single PR generated 350+ comments).

**The service package:**

- **CI/CD Pipeline Hardening** — pin GitHub Actions to commit SHAs, implement dependency scanning, set up Trivy alternatives
- **Supply Chain Incident Response** — for organisations that discover they've been running compromised dependencies
- **Training Workshop** — "Your Developer Tools Are Under Attack" — 2-hour session for engineering teams

**Target market:** UK fintech, healthtech, govtech — any org with developers using VS Code and npm. The Trivy compromise is especially impactful because Trivy is itself a SECURITY tool — the irony makes for an easy sell.

**Revenue Potential:** £3,000-8,000 per audit. Workshop at £2,000/session. Emergency incident response at £5,000-15,000. This is a HOT market right now — every organisation using Trivy (and there are thousands) needs to verify they weren't compromised.
