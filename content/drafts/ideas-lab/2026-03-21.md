## 1. Infrastructure & Automation

### Top Idea: Deploy OpenCode as Open Source Claude Code Alternative

**OpenCode** (460 pts, trending) is a fully open source AI coding agent that hit 120K GitHub stars and 5M monthly devs. Key differentiators vs Claude Code:
- **Free models included** — or connect any provider (Claude, GPT, Gemini, local)
- **GitHub Copilot / ChatGPT Plus login** — use existing subscriptions
- **LSP enabled** — automatically loads the right language servers for the LLM
- **Multi-session** — run multiple agents in parallel on the same project
- **Desktop app + terminal + IDE extension** — available everywhere
- **Share links** — share any session for reference/debugging

**Why this matters:** You already use Claude Code via OpenClaw. OpenCode gives you a zero-cost alternative for less critical coding tasks, or for multi-agent coding workflows where running 3-4 agents simultaneously with paid Claude would be expensive. Route simple tasks to free/local models, reserve Claude for complex work.

**Revenue Potential:** Not direct, but reduces coding tool costs. Could offer "AI-powered development" services using OpenCode + local models on the GPU server for clients who can't use cloud APIs (defence, government, privacy-sensitive).

### Quick Win: Install Sonar CLI for Port Management

**Sonar** (128 pts) — tiny CLI that shows everything listening on localhost with Docker container names, Compose projects, resource usage, and clickable URLs. Kill processes, tail logs, shell into containers — all by port number. Install: `curl -sfL https://raw.githubusercontent.com/raskrebs/sonar/main/install.sh | bash`. Perfect for managing the growing number of services across Mac mini and GPU server. 5 minutes.

### Notable Finds

1. **Attention Residuals (MoonshotAI)** (128 pts) — Drop-in replacement for standard residual connections in Transformers. Each layer selectively aggregates earlier representations via learned, input-dependent attention over depth. Block AttnRes recovers most gains with marginal overhead. Implications for fine-tuning: could improve model quality without changing architecture. Paper + code available.

2. **Sitefire (YC W26)** — "Automating actions to improve AI visibility." If you're doing any content/SEO work, worth watching how they approach AI search optimisation.

3. **Vulkan Compute Shaders in FFmpeg** (151 pts) — Video encoding/decoding using Vulkan compute. Relevant for your RTX 3090s — could dramatically speed up video processing workloads.
