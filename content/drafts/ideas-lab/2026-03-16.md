## 7. Open Source & Community

### Top Idea: MCPScan Unicode/Glassworm Detection Module

The Glassworm invisible Unicode attack is perfect for MCPScan to detect. The attack vector:
1. Invisible Unicode characters (PUA range: 0xFE00-0xFE0F, 0xE0100-0xE01EF) encode payloads
2. A small decoder extracts bytes and passes to `eval()`
3. Looks like an empty string in every editor and code review tool
4. 151+ GitHub repos, multiple npm packages, and VS Code extensions compromised this month

**MCPScan Enhancement:** Add a scanning rule that detects:
- PUA Unicode characters in any MCP server code/config
- The specific decoder pattern (`codePointAt` + PUA range checks)
- eval() calls with seemingly empty template literals
- Known Glassworm indicators from the Aikido writeup

This is a HIGH-VALUE addition because it's a real, active threat that standard code review tools miss. Being the first MCP security scanner to detect invisible Unicode injection = strong differentiator.

**Revenue Potential:** Feature-driven user acquisition for MCPScan Pro

**Quick Win:** Write a regex/AST rule for the PUA Unicode decoder pattern. The signature is distinctive: `w.codePointAt(0)` + `0xFE00` + `0xE0100`. Can be added to MCPScan's rule engine in under an hour.
