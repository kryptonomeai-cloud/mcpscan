# TAYoutube — T A's YouTube Channel Project

**Brand:** T A (personal name as brand)
**Niche:** AI, Cybersecurity, Tech, Hardware, Homelab, Coding
**Tone:** Energetic, hands-on, approachable — like building cool stuff with a mate
**Inspirations:** NetworkChuck, Jakkuh, Gabriel VIP, Jeff Geerling, Fireship

---

## 🎯 Mission

Make AI, cyber, and tech accessible and exciting. Not corporate explainers — real projects, real hardware, real builds. The kind of videos you wish existed when you were figuring this stuff out.

---

## 🏁 Quick Start

1. Read `BRAND.md` — who you are on camera
2. Read `CONTENT-STRATEGY.md` — what to make and when
3. Read `PRODUCTION-GUIDE.md` — how to make it
4. Run `scripts/trend-scanner.sh` — find today's hot topics
5. Pick a video idea → use `templates/video-script-template.md` → film → upload
