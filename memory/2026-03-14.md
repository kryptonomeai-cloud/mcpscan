### Key Learnings
1. **Qwen3 thinking mode**: Always use `"think": false` in Ollama API calls for Qwen3 models
2. **Ollama model swapping**: Pre-warm with `keep_alive: "24h"` to avoid cold-start timeouts
3. **Vercel deploy**: local `next build` success ≠ Vercel deploy success; team permissions can silently block
4. **LoRA DDP vs QLoRA**: Don't try full model replicas on multi-GPU — use QLoRA + balanced sharding
